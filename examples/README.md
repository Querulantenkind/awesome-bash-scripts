# Example Scripts

This directory contains example scripts demonstrating various bash scripting concepts and best practices.

## Available Examples

### hello-world.sh
A simple "Hello World" script demonstrating:
- Basic script structure
- Argument handling
- Color output
- Function usage

**Usage:**
```bash
./hello-world.sh [name]
```

### argument-parsing.sh
Demonstrates various argument parsing techniques:
- Short and long options
- Options with arguments
- Positional arguments
- Input validation

**Usage:**
```bash
./argument-parsing.sh [OPTIONS] [FILES...]
```

## Learning Path

1. Start with `hello-world.sh` to understand basic structure
2. Review `argument-parsing.sh` for handling user input
3. Examine the `templates/script-template.sh` for a complete template
4. Read the documentation in `docs/` for best practices

## Running Examples

Make sure the scripts are executable:
```bash
chmod +x *.sh
```

Then run them:
```bash
./hello-world.sh
./argument-parsing.sh --help
```
